### 🎓 Proyecto EduManager - Caso de Estudio
 #### Descripción de la tarea
 Desarrollar una aplicación web educativa en Java utilizando el ecosistema de Spring que permita gestionar estudiantes y cursos. 
 
 #### Desafío
Construir una aplicación robusta (utilizando Java y el ecosistema de Spring ) que permita separar las responsabilidades administrativas de los usuarios (estudiantes), garantizando que la información sensible sea accesible solo por el personal autorizado. 

#### Solución 
Desarrollé una aplicación web basada en arquitectura de capas (siguiendo el patrón MVC). Implementé **Spring Security** para la autenticación y control de acceso mediante roles, y **Spring Data JPA** para la gestión de persistencia y conexión a la base de datos, utilizando **MySQL** como sistema de almacenamiento de datos. Para la capa de presentación, utilicé **Thymeleaf** complementando con **Javascript, Bootstrap y CSS** para construir una interfaz dinámica. 

#### Herramienta técnicas utilizadas:
Para el **Backend** utilicé **Java, Spring Security, Spring Data JPA y MySQL**.
Para el **FrontEnd** utilicé **Thymeleaf, Javascript, Bootstrap y CSS**

#### Principales aprendizajes
Con este proyecto, aprendí sobre el ecosistema de Spring, amplié mis conocimiento en el lenguaje de Java y mejoré mis habilidades en CSS, Bootstrap y Javascript. Además desarrollé habilidades de gestión del tiempo, priorización de tareas y uso de herramientas de IA con fines educativos, las cuales me ayudaron a comprender conceptos que no había visto o entendido completamente.

#### Métricas de impacto logradas
Durante el desarrollo de EduManager, logré implementar un sistema funcional para la gestión de estudiantes y cursos dentro de un entorno educativo. La aplicación permite realizar operaciones CRUD de forma eficiente, optimizando la administración de la información, utilizando Spring Data JPA.  
Gracias a Spring Security, logré implementar un sistema de autenticación y control de acceso mediante roles, permitiendo el acceso a información sensible solo al personal autorizado. Además integré validaciones básicas en formularios, con el fin de construir una aplicación robusta. Finalmente, al aplicar una arquitectura basada en capas, se facilita la organización del codigo, lo que permitirá una mayor mantenibilidad y escalabilidad.

Todo lo anterior, se traduce en la optimización del tiempo de desarrollo (Gracias a la arquitectura basada en capas), eliminación de vulnerabilidades (gracias a Spring Security y validaciones básicas) y optimización en la carga y persistencia de datos (Gracias a Spring Data JPA).
 
#### Habilidades técnicas aplicadas
- **Java y Spring Boot** para la lógica de negocio y estructura de la aplicación.
- **Spring Security** para implementar autenticación y control de acceso basado en roles.
